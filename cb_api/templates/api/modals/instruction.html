文档说明

URL匹配符

<int: ...>: 正整数，如1

<uuid: ...>: UUID格式，如b7f4b278-6f5d-40a6-b0f0-268814876c4c

<ints: ...>: 逗号分开的正整数，如1,2,3

<uuids: ...>: 逗号分开的UUID，如b7f4b278-6f5d-40a6-b0f0-268814876c4c,20f9af33-d842-446d-aeaa-e78f00537022


URL参数

通过URL传的参数，如a=1&b=2，需要后台开发人员在接口的doc中写明。

对于GET、DELETE接口建议减少URL参数传递，对于PUT、POST接口建议从表单传递数据。


表单字段

只有使用了表单的接口会有表单字段说明。

“-”表示无限制。


搜索

从请求方法，文档说明，URL中搜索。多个关键词之间用空格隔开。


F&Q
F: 接口变更如何记录？

Q: 后台开发人员在接口的Python doc中记录，如
"""
用户登录

2020-01-01 添加remember_me字段
2020-01-02 URL变更：/user/login/ -> /user/login2/
... """

F: 接口为何没有在文档中显示？

Q: 查看接口类属性“is_api”是否为False。

F: 接口为何没有显示表单？

Q: 检查接口类的属性“form_class”是否存在。
