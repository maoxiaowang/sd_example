参数说明
分页
参数名 释义 可选值 示例
pageSize 每页条目数 正整数 pageSize=10（默认10）
page 当前页 正整数 page=1（默认1）
all 显示所有（不分页） true/false all=false（默认false）
过滤
参数名 释义 示例 说明
__exact 相等 name__exact=example 与字段名组合使用
__iexact 相等（忽略大小写） name__iexact=example 同上
__contains 包含 name__contains=example 同上
__icontains 包含（忽略大小写） name__icontains=example 同上
__in 属于 id__in=[1, 2]
name__in=["example1", "example2"] 		列表中可为数字、字符串等
__gt 大于 id__gt=10
created_at__gt=2018-11-28 14:15:52 		过滤日期时间需要数据库的时区支持
__gte 大于等于 id__gte=10 同上
__lt 小于 id__lt=10 同上
__lte 小于等于 id__lte=10 同上
__range 区间 id__range=[1,5]
created_at__range=["2018-11-28 14:15:52","2018-11-28 14:15:53"] 		列表中可为数字、字符串等
__isnull 为空 last_login__isnull=true 匹配数据库中的NULL
exclude__ 排除 exclude__name__exact=example
exclude__id__range=[1,5] 		'exclude__' + 字段名 = 条件
